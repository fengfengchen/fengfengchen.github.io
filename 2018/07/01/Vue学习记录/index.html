Vue学习记录

发表于 2018-07-01

html部分:

内容中 输出内容；
v-？ 表示Vue指令，放在标签头中，用于各种指令
v-bind:/: 绑定一个标签属性，属性值需用引号包裹，可以是一个变量、数组。字符串拼接变量等
v-once 绑定一个变量，只变化一次
v-text 输出变量内容作为标签内容
v-html 区别在于可以转化HTML格式
v-model 双向绑定变量，变量值改变，会反应在标签中，标签内容改变，也会改变变量值
v-for 循环指令
v-if/v-else v-if/v-else-if…/v-else 条件判断
v-show 控制显示隐藏 效果比v-if要好点

script部分

var app = new Vue({}); 实例化Vue对象，内容用属性的键值对表示
el 表示一个实例化对象，值一般是#ID，字符串
data 存放变量数据的地方，本身是一个对象，对象中的元素值可以是字符串，对象，数组之类的
computed 计算属性

组件

props: 子组件的属性变量存放，通常会将父组件的值作为属性值传给子组件
slot 内容分发，在模板中定义slot标签的位置，在渲染过程中会将实际标签将其替换，有一个name属性，可以定义实际渲染的部分
scrop: 将子组件传递过来的props对象传递给一个临时变量，然后可以读取该临时变量的一些值
