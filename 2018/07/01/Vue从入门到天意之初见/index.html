Vue从入门到天意之初见

发表于 2018-07-01 | 更新于 2018-10-26

一些废话

Vue 是一套用于构建用户界面的渐进式框架。嗯！
本人断断续续学习Vue也有一段时间了，只是一直没啥进展，而且很容易就忘了。故决定从头开始，认真学习。特来记录学习的过程，以铭吾志！

生命周期:

每个 Vue 应用都是通过用 Vue 函数创建一个新的 Vue 实例开始的

下面是Vue实例的生命周期

解释一下生命周期就是Vue实例（将简称实例）在什么情况下会发生是什么事情的表示。

Vue语法

可以认为Vue语法由这三块内容组成 插值，指令和实例属性

插值

Vue.js 使用了基于 HTML 的模板语法，允许开发者声明式地将 DOM 绑定至底层 Vue 实例的数据。
简单来说即使可以直接使用双大括号的形式 进行文本插值，且可以使用 javascript 表达式

指令

指令 (Directives) 是带有 v- 前缀的特殊特性。可以自定义指令，将在以后讨论；

指令一般的形式是这样的 v-xx=’oo’
常见的比如：

  1. 用于条件判断的指令
    v-if/v-else v-if/v-else-if…/v-else & v-show
  2. 循环数组或者对象的指令
    v-for
    特殊一点的指令
    v-bind:特性=’变量’ v-bind指令用于绑定html特性，可以动态更新html特性的值，可以简写成 :
    v-on:event=’函数’ v-on指令用于绑定事件，用于监听事件的触发，可以简写成 @
    v-model 用于表单的双向数据绑定，可以等价于 v-bind:value，v-on:input

指令可以添加修饰符，用于指出一个指令应该以特殊方式绑定。

实例属性

el 表示一个实例化对象，值一般是#ID，字符串，如果不存在template属性，将以这个这个作为模板
template 模板属性，实例的渲染模板，优先级比el要高，这是由生命周期决定的
data 数据属性，存放初始数据，是一个对象
computed 计算属性，可以用于替代插值中的复杂计算，默认只有 getter ，不过在需要时你也可以提供一个 setter
methods 方法属性，用于处理一些逻辑，不同与计算属性是基于它们的依赖进行缓存的
watch 侦听属性，观察和响应 Vue 实例上的数据变动，用于需要在数据变化时执行异步或开销较大的操作
