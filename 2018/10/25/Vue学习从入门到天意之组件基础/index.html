Vue从入门到天意之组件基础

发表于 2018-10-25

组件基础

组件是可复用的 Vue 实例，所以组件可以与new vue接收相同的选项，如data,computed等以及生命周期的钩子。仅有的例外是像 el 这样根实例特有的选项；
组件带有一个名字；
组件可以全局注册，也可以局部注册；
需要注意的是：

  1. data属性必须是一个函数，通常是函数返回一个对象
  2. 每个组件有且只有一个根元素

组件的通信方式

我们一般可以这么理解，Vue实例直接放置组件实例，故应为父组件，而我们创建的组件实例放置在Vue实例之中，故应为子组件。
那么父子组件之间应该应该如何传递数据呢？

可以用一句话概括：
子组件利用prop属性，自定义一些html属性用于获得父组件的数据；
子组件利用自然事件，触发绑定在父组件上的自定义事件，向父组件传递消息；
